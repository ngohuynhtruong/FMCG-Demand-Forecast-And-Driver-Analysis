## Project Overview
This project analyzes FMCG sales data to identify key demand drivers 
and build a demand forecasting model to support business planning.

## Key Analysis
- Sales trend by time, SKU, and region
- Impact of promotion and seasonality
- Driver analysis for units sold
- Demand forecast using Machine Learning

## Tools
- Python (Pandas, NumPy, XGBoost, LightGBM)
- Data Visualization (Matplotlib, Seaborn)
